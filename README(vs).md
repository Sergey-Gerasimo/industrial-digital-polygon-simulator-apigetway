# Industrial Digital Polygon Simulator API Gateway

API Gateway для индустриального симулятора на базе FastAPI.

# Функциональность

- Аутентификация и авторизация (JWT)
- Управление пользователями
- Управление игровыми комнатами
- Система приглашений
- Мониторинг (Prometheus метрики)
